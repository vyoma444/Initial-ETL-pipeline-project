# ETL Pipeline using Python and SQLite

## Overview
This project implements an ETL pipeline using the Superstore dataset.

### Features
- Extract data from CSV
- Transform data into star schema
- Data quality checks
- Load into SQLite warehouse
- Revenue verification query

### Technologies
- Python
- Pandas
- SQLite

### Run
```bash
python etl_pipeline.py
```
